1. Karena Visual Studio Code Gratis

2. Di dalam VS Code sudah terinstall plugin Emmet

3. Di dalam VS Code sudah terinstall Linter(Fungsi utk cek codingan benar atau

   sarannya apa yg mau kita ketikkan

5. Kemudahan bekerja dgn warna

6. Sidebar yg kaya akan fitur

7. Mudah mengelola Extensions/Plugin

8. Terintegrasi dgn Git
